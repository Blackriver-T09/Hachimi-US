# 自动播放故障排除

## 🔍 问题：自动播放不工作

### 现象
- ✅ 底部显示播放器
- ✅ 选择的歌曲正确
- ❌ 但音频没有自动开始播放
- 显示暂停按钮 ⏸ 而不是播放按钮 ▶

---

## 🛡️ 原因：浏览器自动播放策略

现代浏览器为了保护用户体验，会阻止未经用户交互的自动播放。

### Chrome/Edge 策略
- ❌ 阻止：页面加载时自动播放有声音频
- ✅ 允许：用户点击页面后自动播放
- ✅ 允许：静音状态下自动播放

### Firefox 策略
- ✅ 默认允许自动播放
- 可在设置中修改

### Safari 策略
- ❌ 严格阻止自动播放
- 需要明确的用户交互

---

## ✅ 解决方案

### 方案 1: 点击页面任意位置（推荐）

**最简单的方法：**
1. 刷新页面
2. 看到播放器出现后
3. **点击页面任意位置**
4. 音乐会立即开始播放

### 方案 2: 允许自动播放（Chrome）

**Chrome 浏览器设置：**
1. 点击地址栏左侧的 🔒 或 ⓘ 图标
2. 点击"网站设置"
3. 找到"声音"选项
4. 设置为"允许"
5. 刷新页面

**或者使用 Chrome 标志：**
```
chrome://settings/content/sound
```
添加 `http://localhost:5173` 到允许列表

### 方案 3: 启动时静音（未来功能）

可以修改为启动时静音播放，用户点击后取消静音：
```typescript
// 静音自动播放
audioRef.current.muted = true
audioRef.current.play()
```

---

## 🧪 测试自动播放是否被阻止

### 打开浏览器控制台
1. 按 F12 打开开发者工具
2. 切换到 Console 标签
3. 刷新页面
4. 查看是否有以下消息：

```
Auto-play blocked by browser, waiting for user interaction
```

如果看到这条消息，说明浏览器阻止了自动播放。

## 🎯 当前实现逻辑

```typescript
// 1. 设置当前视频
setCurrentVideo(initialVideo)
setIsPlaying(true)

// 2. 尝试自动播放
setTimeout(() => {
  if (audioRef.current) {
    audioRef.current.play().catch(_err => {
      console.log("Auto-play blocked by browser")
      // 等待用户交互
    })
  }
}, 100)
```

### 工作流程
```
页面加载
    ↓
获取配置 (START_INDEX = 7)
    ↓
设置 currentVideo 和 isPlaying
    ↓
尝试自动播放
    ↓
┌─────────┴─────────┐
│                   │
成功              被阻止
│                   │
开始播放          等待用户点击
显示 ⏸           显示 ⏸
                    ↓
                用户点击页面
                    ↓
                开始播放
```

## 🔧 开发者选项

### Chrome 禁用自动播放策略（仅开发）

**启动 Chrome 时添加标志：**
```bash
google-chrome --autoplay-policy=no-user-gesture-required
```

**或：**
```bash
chromium --autoplay-policy=no-user-gesture-required http://localhost:5173
```

⚠️ **注意：** 这仅用于开发测试，不影响最终用户体验。

## 📝 总结

### 问题
浏览器阻止自动播放以保护用户体验

### 解决
点击页面任意位置即可开始播放

### 原理
```javascript
// 浏览器要求用户交互后才能播放
document.addEventListener('click', () => {
  audioRef.current.play() // ✅ 允许
})

// 页面加载时直接播放
audioRef.current.play() // ❌ 被阻止
```

### 最佳实践
1. 刷新页面
2. 点击任意位置
3. 开始播放
